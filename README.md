## 🛠️ Herramientas y Validación
Para que el código sea profesional y pase las pruebas del profesor, usé:
* **Bootstrap 5:** Para que la página se vea bien en celulares y computadoras (Responsive).
* **BEM:** Una forma ordenada de nombrar las clases en el CSS.
* **Validación HTML5/CSS3:** El formulario funciona usando solo reglas de HTML y CSS.
* **W3C & Stylelint:** Herramientas automáticas de GitHub que verifican que el código no tenga errores de escritura.
